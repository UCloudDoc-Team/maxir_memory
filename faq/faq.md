# 常见问题解答

## 产品与功能

**Q：MAXIR Memory 支持哪些 AI 工具？**

MAXIR Memory 通过 MCP（Model Context Protocol）协议与 AI 工具集成，目前支持任何实现了 MCP 客户端的 AI 工具，包括 Claude Desktop、Cursor 等，以及通过自定义 Agent 开发接入的场景。


---

**Q：MAXIR Memory 如何保证数据安全？**

MAXIR Memory 对传输层使用 TLS 1.3 加密，静态数据使用 AES-256 加密存储。平台通过 ISO 27001、SOC 2 Type II 认证。如需了解合规详情，请联系 UCloud 支持。

---

**Q：我的数据可以导出吗？**

可以。通过 API 可以导出记忆数据。管理员可以通过管理后台查看项目和文档数据。

---

## 使用与操作

**Q：文件上传后多久可以被 AI 检索？**

大多数文件（PDF、Word 等文档类文件）的索引处理通常在几分钟内完成，具体时间取决于文件大小和内容复杂度。文件状态从"索引中"变为"就绪"后即可被 AI 检索。

---

**Q：文件大小有限制吗？**

单个文件最大支持 500 MB。

---

**Q：MCP API Key 泄露了怎么办？**

立即在 MAXIR Memory 控制台的项目 MCP 服务器页面，找到对应 Key 并点击**撤销**，Key 将立即失效。然后重新生成新的 API Key，更新相关 AI 工具的配置。

---


## 获取帮助

**Q：遇到技术问题如何联系支持？**

通过控制台提交工单，选择产品类别 **MAXIR Memory**，详细描述问题现象和复现步骤（如有报错信息，请附上完整的错误信息和实例 ID）。工单将由技术支持团队跟进处理。
