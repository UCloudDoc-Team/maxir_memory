# 产品优势

## 极致数据规模扩展能力

相比直接将文件喂给 LLM，MAXIR Memory 通过向量化存储与离线索引技术，支持 **10,000 倍**的数据规模扩展，同时保持毫秒级检索延迟。

- **突破上下文窗口**：文档知识库不受 LLM 上下文窗口限制，支持检索海量文档
- **近乎完美的召回率**：生产环境验证的 100M+ 复杂混合文档场景下极高记忆召回率
- **大幅节省 Token**：与直接 LLM 文件读取相比，显著降低 Token 消耗
- **毫秒级响应**：向量检索响应在毫秒级，不影响 AI 交互体验

## 完整的记忆溯源

MAXIR Memory 为每条通过 API 提取的记忆维护完整的来源链路和变更历史：

- **来源追踪**：每条记忆可追溯到触发提取的原始对话消息
- **版本历史**：记录每次记忆变更（ADD / UPDATE / DELETE）的完整历史
- **事件关联**：每个版本与触发该变更的对话事件绑定，全程可查

## MCP 协议原生支持

MAXIR Memory 原生支持 Model Context Protocol（MCP），允许 Claude Desktop、Cursor 等支持 MCP 的 AI 工具通过标准化协议安全访问您的项目知识库，无需修改现有工具代码。

每个项目独立配置 MCP 端点和 API Key，访问范围严格限定在项目内，外部 AI 只能读取，无法修改或删除数据。

## 丰富的多模态文档支持

MAXIR Memory 支持多种格式文件的处理和索引：

| 数据类型 | 支持格式 |
|----------|----------|
| 文档 | PDF、Word（.docx/.doc）、TXT、Markdown、JSON |
| 表格 | Excel（.xlsx/.xls）、CSV |
| 其他 | 更多格式持续支持中 |

## 灵活的外部文档接入（即将推出）

连接器功能即将上线，届时可直接接入 WPS（金山文档）等外部文档服务，无需手动下载和重新上传文件，即可在 MAXIR Memory 中浏览和使用外部文档，并与项目关联。
