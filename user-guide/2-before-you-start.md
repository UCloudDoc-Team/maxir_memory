# 开始之前

## 前提条件

在使用 MAXIR Memory 之前，请确认：

1. 您的组织已在 UCloud 购买了 MAXIR Memory 实例，且实例状态为**运行中**。
2. 管理员已为您创建账号，并告知您以下信息：
   - **登录地址**：用户入口地址（由管理员提供，可能是内网或外网地址）
   - **账号**：您的用户名或邮箱
   - **初始密码**：首次登录后建议立即修改密码

## 登录系统

1. 在浏览器中访问管理员提供的用户入口地址。
2. 输入账号和密码，点击**登录**。
3. 首次登录建议修改初始密码：点击右上角头像 > **个人设置** > **修改密码**。

## 界面概览

登录后，您将看到 MAXIR Memory 的主界面，主要区域说明如下：

| 区域 | 说明 |
|------|------|
| **左侧导航栏** | 功能入口，包括文档管理、项目管理、连接器等 |
| **顶部工具栏** | 通知、个人设置等全局操作 |
| **主内容区** | 当前功能的主要操作界面 |

## 核心概念速查

在开始使用前，了解以下核心概念有助于更好地理解产品：

### 文档驱动（Document Drive）

MAXIR Memory 提供个人云端文档空间（My Space），支持上传 PDF、Word、Excel、CSV、Markdown、TXT 等格式的文件，并可用层级文件夹组织管理。

### 项目（Project）

项目是组织文档和记忆的逻辑单元。通过将相关文档和记忆归组到同一个项目，可以为 AI 提供聚焦的上下文。每个项目可以配置独立的 MCP 服务器端点。

### MCP 服务器

MCP（Model Context Protocol）是 AI 模型访问外部数据的标准协议。MAXIR Memory 为每个项目生成独立的 MCP 端点，允许 Claude Desktop 等支持 MCP 的 AI 工具安全访问您的项目数据。
