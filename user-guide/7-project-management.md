# 项目管理

项目（Project）是 MAXIR Memory 中组织文档和记忆的核心单元。通过将相关文档和知识归组到一个项目，您可以为 AI 提供聚焦的上下文，显著提升问答质量和响应准确性。

## 项目概览

每个项目包含：
- **关联文档**：从文档库中选择与该项目相关的文件
- **文字记忆**：存储在项目中的知识片段和指导信息
- **MCP 服务器**：向外部 AI 工具提供项目数据访问的端点

## 创建项目

1. 在左侧导航栏，点击**项目管理**。
2. 点击**新建项目**。
3. 填写以下信息：

   | 字段 | 是否必填 | 说明 |
   |------|----------|------|
   | 项目名称 | 必填 | 易于识别的名称，建议体现项目主题 |
   | 项目描述 | 可选 | 简要说明项目内容和用途，帮助团队成员快速理解 |

4. 点击**确定**，项目创建完成，自动跳转至项目详情页。

## 项目详情页

项目详情页分为以下标签页：

| 标签 | 功能 |
|------|------|
| **概览** | 查看项目统计信息（文档数、记忆数、MCP 状态） |
| **文档** | 管理项目关联的文档 |
| **记忆** | 管理项目内的文字记忆 |
| **MCP 服务器** | 配置和管理项目的 MCP 访问端点 |

## 管理项目文档

### 添加文档到项目

1. 进入项目详情页，点击**文档**标签。
2. 点击**添加文档**。
3. 在弹出的文件树选择器中，勾选要关联的文件或文件夹（支持多选）。
4. 点击**确认**，选中的文件将出现在项目文档列表中。

> 添加文件夹时，该文件夹及其所有子文件将同时被关联到项目。

### 从项目移除文档

1. 在项目文档列表，找到目标文档。
2. 点击右侧**移除关联**。

移除关联不会删除文档本身，文档仍在文档库中保留。

### 文档索引状态

关联到项目的文档会自动建立向量索引，索引完成后可被 AI 检索。状态说明：

| 状态 | 说明 |
|------|------|
| **索引中** | 正在处理文档内容，建立向量索引 |
| **就绪** | 索引完成，文档可被 AI 检索 |
| **处理失败** | 索引失败，可点击**重试**再次尝试 |


### 添加记忆

1. 进入项目详情页，点击**记忆**标签。
2. 点击**添加记忆**。
3. 输入记忆内容（支持 Markdown 格式）。
4. 点击**保存**。

**记忆内容示例：**

```markdown
## 项目背景
本项目收录了 2024 年公司年度报告及季度财报。

## 使用说明
- 回答财务相关问题时，请优先引用报告中的具体数据
- 引用时请注明报告名称和章节
- 涉及预测数据请标注为"预测值"
```

### 编辑和删除记忆

1. 在记忆列表，点击目标记忆右侧的**编辑**，修改内容后保存。
2. 点击**删除**可永久删除该条记忆。

## 编辑和删除项目

### 编辑项目名称/描述

1. 在项目详情页，直接点击项目名称或描述，进行内联编辑。
2. 编辑完成后按回车或点击空白处保存。

### 删除项目

1. 在项目列表，找到目标项目，点击右侧**更多操作** > **删除项目**。
2. 确认删除操作。

> **注意**：删除项目将同时删除项目内的所有记忆和 MCP 配置，但不会删除关联的文档（文档仍保留在文档库）。此操作不可撤销。
