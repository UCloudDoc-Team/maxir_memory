# 文档管理

MAXIR Memory 提供完整的云端文档管理能力，支持将各类格式的文件上传、组织并与 AI 项目关联，形成可被 AI 检索的知识库。

## 支持的文件格式

| 类别 | 格式 |
|------|------|
| 文档 | PDF、Word（.docx、.doc）、TXT、Markdown（.md）、JSON |
| 演示 | PowerPoint（.pptx） |
| 表格 | Excel（.xlsx、.xls）、CSV |
| 图片 | PNG、JPG/JPEG、GIF、WebP、BMP |
| 音频 | MP3、WAV、M4A、AAC |
| 视频 | MP4、MOV、AVI、MKV |

## 上传文档

### 上传单个文件

1. 在左侧导航栏，点击**文档管理**。
2. 点击**上传文件**按钮，或直接将文件拖拽到文件区域。
3. 在文件选择对话框中，选择要上传的文件，点击**打开**。
4. 文件上传后，系统自动开始处理（提取文本、建立向量索引）。
   - 处理中：状态显示**处理中**
   - 处理完成：状态变为**就绪**，文件可被 AI 检索

### 批量上传文件夹

1. 点击**上传文件夹**，选择本地文件夹。
2. 系统将保留文件夹的层级结构，批量上传所有文件。
3. 可在上传过程中查看进度，大量文件上传可能需要数分钟。

### 上传限制

| 限制项 | 说明 |
|--------|------|
| 单文件大小 | 最大 500 MB |
| 单次批量上传文件数 | 最多 100 个文件 |
| 总存储配额 | 依规格版本而定，见实例详情 |

## 文件组织

### 创建文件夹

1. 在文档管理页面，点击**新建文件夹**。
2. 输入文件夹名称，点击**确认**。
3. 文件夹支持无限层级嵌套，可按项目、主题、时间等维度组织文件。

### 移动文件或文件夹

1. 右键点击目标文件或文件夹，选择**移动到**。
2. 在文件树中选择目标位置，点击**确认移动**。

### 重命名

1. 右键点击文件或文件夹，选择**重命名**。
2. 输入新名称，按回车或点击确认。

### 删除

1. 右键点击文件或文件夹，选择**删除**。
2. 确认删除操作。

> **注意**：删除文件夹将同时删除其中的所有文件，且操作不可撤销。如果文件已关联到项目，删除后该关联也将同步移除。

## 文档详情与注释

### 查看文档详情

1. 点击文档列表中的文件名，进入文档详情页。
2. 详情页显示：
   - 文件基本信息（大小、格式、上传时间、处理状态）
   - 文档内容预览（支持 PDF、图片等格式的在线预览）
   - 关联的项目列表

### 添加注释

支持对文件和文件夹添加 Markdown 格式的注释，用于记录文档的背景信息、使用说明或重要提示。

1. 在文档详情页，点击**添加注释**。
2. 使用 Markdown 格式输入注释内容。
3. 点击**保存**。

注释内容将作为文档的附加上下文，与文档正文一同被 AI 检索和引用。

## 文档与项目关联

### 将文档关联到项目

1. 进入**项目管理**，点击目标项目。
2. 在项目详情页，切换到**文档**标签，点击**添加文档**。
3. 在文件树中选择要关联的文件或文件夹（支持多选）。
4. 点击**确认关联**。

### 取消关联

1. 在项目的文档列表中，找到目标文档。
2. 点击右侧的**取消关联**按钮。

取消关联不会删除文件本身，只是移除文件与该项目的关联关系。

## 搜索与筛选

### 全文搜索

在文档管理页面顶部的搜索框中输入关键词，可对文档名称和内容进行全文搜索。

### 筛选

支持按以下条件筛选文档：

- **格式**：按文件类型筛选
- **状态**：按处理状态筛选（处理中 / 就绪 / 处理失败）
- **上传时间**：按时间范围筛选
- **关联项目**：筛选特定项目关联的文档

## 下载文档

1. 在文档列表，右键点击目标文件，选择**下载**。
2. 或进入文档详情页，点击顶部**下载**按钮。
