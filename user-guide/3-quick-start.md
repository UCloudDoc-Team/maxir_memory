# 快速入门

本文引导您在 10 分钟内完成 MAXIR Memory 的首次配置，上传文档并体验 AI 检索能力。

## 前提条件

- 已完成登录，参考[开始之前](/maxir_memory/user-guide/2-before-you-start.md)。
- 已准备好至少一个文档文件（PDF、Word、Excel、CSV、Markdown 等）。

---

## 步骤 1：上传文档

1. 在左侧导航栏，点击**文档管理**，进入个人云盘（My Space）。
2. 点击**上传**按钮，选择本地文件（支持 .md、.txt、.csv、.pdf、.xlsx、.xls 格式）。
3. 可选择多个文件，也可上传整个文件夹（保留文件夹层级结构）。
4. 上传完成后，文件出现在文件树中，系统自动开始处理索引。

---

## 步骤 2：创建项目

1. 在左侧导航栏，点击**项目管理**。
2. 点击**创建项目**。
3. 填写**项目名称**（例如"Q1 研究资料"）和可选的**项目描述**。
4. 点击**保存**。

创建成功后，项目卡片显示文档数量、记忆数量和 MCP 状态。

---

## 步骤 3：关联文档到项目

1. 点击刚创建的项目，进入项目详情页。
2. 切换到**文档**标签，点击**添加文档**。
3. 在文件树中，用复选框选择要关联的文件或文件夹。
4. 点击**确认**。

关联的文档状态从 `pending` → `running` → `okay`，就绪后即可被 AI 检索。

---

## 步骤 4：与目标应用集成

1. 进入到**集成**页面，选择项目，再选择要与MAXIR Memory进行记忆连接的目标应用。
2. 选择对应的API Key，若没有API Key，可一键新建。
3. 参考控制台展示的**接入方式**，按指导完成安装配置。
